PASOS RÁPIDOS:
1. Google Cloud Console → Create Service Account → Download JSON
2. Google Analytics → Property Access → Add Service Account email
3. Railway → Add environment variables (GA_PRIVATE_KEY, GA_CLIENT_EMAIL)
4. Backend → npm install @google-analytics/data
5. Actualizar código (ver GUIA-GOOGLE-ANALYTICS.md)
