INSTRUCCIONES PARA USAR TIPOGRAFÍAS LOCALES EN ELEMENTOR
=========================================================

Si tienes las tipografías Soulcraft e IBM Plex Mono descargadas en tu ordenador, 
hay 2 formas de usarlas en Elementor:

OPCIÓN 1: SUBIR LAS FUENTES A WORDPRESS (RECOMENDADO)
------------------------------------------------------
1. Sube los archivos de fuentes (.woff2, .woff, .ttf) a tu WordPress:
   - Ve a Medios → Añadir nuevo
   - Sube los archivos de las tipografías

2. En el HTML del código de precios, reemplaza estas líneas:

   BUSCA (línea 8-19):
   @import url('https://fonts.googleapis.com/css2?family=IBM+Plex+Mono:wght@400;500;600;700&display=swap');
   @import url('https://fonts.cdnfonts.com/css/soulcraft');

   REEMPLAZA POR:
   @font-face {
       font-family: 'IBM Plex Mono';
       src: url('URL_DE_TU_WORDPRESS/IBMPlexMono-Regular.woff2') format('woff2');
       font-weight: 400;
       font-style: normal;
   }
   @font-face {
       font-family: 'IBM Plex Mono';
       src: url('URL_DE_TU_WORDPRESS/IBMPlexMono-Bold.woff2') format('woff2');
       font-weight: 700;
       font-style: normal;
   }
   @font-face {
       font-family: 'Soulcraft';
       src: url('URL_DE_TU_WORDPRESS/Soulcraft.woff2') format('woff2');
       font-weight: normal;
       font-style: normal;
   }

OPCIÓN 2: USAR PLUGIN DE FUENTES PERSONALIZADAS
------------------------------------------------
1. Instala el plugin "Custom Fonts" o "Use Any Font"
2. Sube tus tipografías desde el panel del plugin
3. El plugin generará automáticamente el CSS necesario
4. Las fuentes estarán disponibles en todo WordPress/Elementor


OPCIÓN 3: VERIFICAR SI LAS FUENTES CARGAN DESDE CDN
----------------------------------------------------
Si el código actual no muestra las fuentes correctamente:

1. Abre el navegador y ve a la página
2. Presiona F12 (Consola de Desarrollador)
3. Ve a la pestaña "Network" o "Red"
4. Recarga la página
5. Busca errores en la carga de fuentes (archivos .woff2 o .woff)

Si hay errores, significa que Elementor está bloqueando las fuentes externas.


FALLBACKS INCLUIDOS
-------------------
He añadido fuentes de respaldo (fallback) para que si las tipografías 
personalizadas no cargan, se usen fuentes del sistema similares:

- Soulcraft → Impact → Arial Black
- IBM Plex Mono → Courier New → Courier

Esto garantiza que el diseño se vea bien aunque las fuentes no carguen.


NOTA IMPORTANTE PARA ELEMENTOR
-------------------------------
Al pegar el código en un widget HTML de Elementor:
- Pega TODO el contenido del archivo pricing-section.html
- Incluye las etiquetas <style> y <script>
- NO uses el editor de código de WordPress (puede romper el formato)
- Usa directamente el widget HTML de Elementor
